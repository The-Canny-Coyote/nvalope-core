Installing Nvalope on your device

Progressive Web App (PWA) · Your budget data stays on your device

What is a PWA?

A Progressive Web App is a website that your browser can install so Nvalope opens from your home screen or app list, often in its own window, similar to an app from an app store. It is still the same Nvalope site; nothing extra is uploaded to us.

After the first successful load, core parts of the app can work offline (depending on your browser). Updates arrive when you use the app online, like other web apps.

Why install?

  • Quick access from an icon instead of typing the address every time.
  • Can feel more like a dedicated app (full screen or separate window on some systems).
  • Matches how Nvalope is designed: local-first, no account required.

Before you start

Use a supported browser. Installation is built into Chrome, Edge, and Safari on mobile; Firefox support varies by platform. You must visit Nvalope in the browser first (not inside another app’s built-in browser if that blocks install).

iPhone or iPad (Safari)

  1. Open Nvalope in Safari (Apple does not allow other browsers to install PWAs the same way).
  2. Tap the Share button (square with an arrow pointing up).
  3. Scroll and tap Add to Home Screen.
  4. Edit the name if you like, then tap Add.

If you don’t see “Add to Home Screen,” scroll the share sheet or tap “Edit Actions” and enable it.

Android (Chrome)

  1. Open Nvalope in Chrome.
  2. When offered, tap Install or Add to Home screen; or open the menu (⋮) and choose Install app or Add to Home screen.
  3. Confirm the install.

Samsung Internet and other Chromium-based browsers often have a similar “Add to Home screen” or “Install” option in their menu.

Windows or macOS — Chrome or Edge

  1. Open Nvalope in Chrome or Microsoft Edge.
  2. Look for an install icon in the address bar (often a computer with a down arrow, or a plus sign) and click it; or open the browser menu (⋮) → Install Nvalope / Apps → Install this site as an app.
  3. Confirm. Nvalope may open in its own window and appear in your Start menu (Windows) or Applications (macOS Chrome).

Windows or macOS — Firefox

Firefox can install supported sites as a shortcut or app depending on version. Open the main menu (☰) and look for Install, or use the page actions in the address bar if shown. If install is not offered, use Chrome or Edge for a one-click PWA install.

macOS — Safari

On macOS, Safari does not install PWAs the same way as on iPhone. Use Chrome or Edge to install Nvalope on the desktop, or add a Dock shortcut: File → Add to Dock for a web shortcut (behavior may differ from a full PWA install in Chrome).

Linux

Chromium-based browsers (Chrome, Edge, many system browsers) follow the same pattern as Windows: install icon or menu → install as app. Firefox may offer a shortcut depending on your distribution.

After installing

  • Launch Nvalope from the new icon like any other app.
  • Your data remains in the browser’s storage on that device; backups and exports still work as described in the app.
  • To remove the install: uninstall like any other app (platform-specific), or remove the shortcut from your home screen.
